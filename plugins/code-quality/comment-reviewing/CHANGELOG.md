# Changelog

All notable changes to the Comment Reviewer skill will be documented in this file.

## [1.0.0] - 2025-01-21

Initial release.

### Features
- Comment review and improvement following "why not what" principle
- Five-category action system: Remove, Improve, Keep, Flag, Condense
- Intelligent comment type detection (implementation vs API documentation)
- Multi-language support (C-style, Python, SQL, HTML, etc.)
- Git integration for reviewing changed files
- Project-specific configuration via `.reviewrc.md`
- Code-comment consistency checking
